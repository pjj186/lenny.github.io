# DDL이란?

DDL(Data Definition Language)는 데이터베이스의 구조를 정의하고 관리하는 SQL 명령어들을 말합니다.

## DDL의 주요 명령어

### CREATE

새로운 데이터베이스 객체를 생성합니다.

### ALTER

기존 데이터베이스 객체의 구조를 변경합니다.

### DROP

데이터베이스 객체를 삭제합니다.

### TRUNCATE

테이블의 모든 데이터를 삭제합니다.

## DDL의 특징

DDL 명령어는 실행 즉시 데이터베이스에 반영되며, 롤백이 불가능합니다.

### 자동 커밋

DDL 명령어는 자동으로 커밋됩니다.

### 스키마 변경

데이터베이스의 스키마 구조를 변경할 수 있습니다.
